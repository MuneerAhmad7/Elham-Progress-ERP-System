## ✨ Features

### 🏢 Multi-Module ERP System

- **HR & HCM** - Complete human capital management
- **Finance & Accounting** - Full accounting with ZATCA e-invoicing
- **CRM** - Customer relationship management
- **Projects** - Construction project management
- **Tasks** - Advanced task & work management

### 🇸🇦 Saudi Compliance

- ZATCA E-Invoicing (Phase 1 & 2)
- GOSI Integration
- Qiwa Integration
- Mudad/WPS Compliance
- Hijri Calendar Support
- Arabic/English Bilingual

### 🎨 Modern UI/UX

- Responsive design (mobile-first)
- Dark mode support
- RTL/LTR support
- Progressive Web App (PWA)
- Accessible (WCAG 2.1 AA)

### 🔐 Security

- JWT Authentication
- Role-based Access Control (RBAC)
- Multi-factor Authentication (MFA)
- Audit logging
- Data encryption

## 🛠️ Tech Stack

### Frontend

- **Framework:** Vue.js 3 (Composition API)
- **Build Tool:** Vite
- **State Management:** Pinia
- **Routing:** Vue Router 4
- **UI Framework:** Tailwind CSS + DaisyUI
- **Charts:** ApexCharts
- **Icons:** Heroicons
- **HTTP Client:** Axios
- **Internationalization:** Vue I18n

### Backend (Separate Repo)

- Python Flask/FastAPI
- PostgreSQL
- Redis
- Celery
